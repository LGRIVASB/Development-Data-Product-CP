Instructions

The app has 3 section. The first one is the general description and instruction tab. The second one is a data exploration tab where you can explore 3 different dataset and the last tab is model visualization there you can build a linear regression model.

Data Exploration

This section you can explore the datasets. In the selection panel, you can choose the dataset and the number of the observation that will display the application. Then clic on the View Data button to refresh the visualization.

The main panel is going to show the general description of the selected data, the summary and a table with the number of observation that you want to see.

The available dataset are:

  • mtcars
  • iris
  • swiss

Model Visualization

Here you will have a selection panel that has 3 drop-down list. To create your model follow the next steps:

  1. Select the dataset that you want to use
  2. Select the predictive/independent variable
  3. Select the outcome/dependent variable.

The main panel is going to display the summary of the model with the variables that you selected before. Additionally, the table of the coefficient of the fitted model and the scatter plot with a red line which is the linear regression model.
